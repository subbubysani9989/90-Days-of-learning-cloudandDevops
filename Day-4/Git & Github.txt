Commit : 
working directory > Stage >commit> local repository

Remove : 

git rm filename (it deletes and stages )
git reset --hard (birng back  the deleted files )
git rm -f filename (forcefully delete the unstaged changes)
git rm -r filename/directory (to remove the Recursively)

checking the commit history : 

git log

Branching : 

git branch (to create the branches)
git checkout (to change the branches)
git merge development -m "message" (to merge from development to main)

Merge Conflicts : 
git merge staging (if we have same content inside the same branches)
git merge development
